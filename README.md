# Shopping-Cart

A React based online shopping web app based on The Odin project React course.  
The web app uses [Fake store API](https://fakestoreapi.com/) to populate the items. It has features of adding to cart, incrementing and decrementing number of item selected, filtering items based on available options and dynamic price calculation.

[Live Demo](https://yidnekachewsk-shopping-cart.netlify.app/)
